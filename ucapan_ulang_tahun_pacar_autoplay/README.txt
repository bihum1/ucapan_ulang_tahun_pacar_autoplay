CARA PAKAI:
1. Ganti foto di folder assets/img dengan nama:
   foto1.jpg, foto2.jpg, foto3.jpg, foto4.jpg
2. Ganti musik di folder assets/music dengan nama:
   lagu.mp3
3. Untuk GitHub Pages, upload semua isi folder ini ke repository GitHub.
4. Aktifkan GitHub Pages dari Settings > Pages > Deploy from branch > main > /root.

CATATAN:
- Musik otomatis biasanya perlu klik tombol terlebih dahulu karena aturan browser.
- Jangan ubah nama folder assets kecuali juga mengubah path di index.html.
